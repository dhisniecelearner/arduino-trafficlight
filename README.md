# Arduino Traffic Light Simulation

## Overview
This project simulates a basic traffic light system using Arduino Uno in Tinkercad.

## Components Used
- Arduino Uno
- Red LED
- Yellow LED
- Green LED
- 220Ω Resistors
- Breadboard
- Jumper Wires

## Working
The LEDs glow in the following sequence:
1. Red
2. Yellow
3. Green
4. Repeat continuously
